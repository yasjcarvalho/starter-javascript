# Tipos de dados

Conforme o ECMAScript standard temos 9 tipos de dados:

- Data types
- Primitive / Primitive value
- Structural
- Structural Primite

## Primitivos

- String "ABC" "BCA" "bca" "Bca" "bcA"
- Number: 22 23 21 40
- Boolean: true / false
- undefined
- Symbol
- BigInt

### Estruturais

- Object
  - Array
  - Map
  - Set
  - Date
  - ...
- Function

#### Primitivo Estrutural / Structural Root Primitive

- null
